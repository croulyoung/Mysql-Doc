DB Doc - document your databases automatically

SQL Doc is a fast, simple tool which automatically generates database documentation.

You can create documents in HTML, Microsoft Word or Compiled HTML Help files. Information about object definitions and dependencies is automatically included, and you can add further descriptions to your database objects if necessary.

SQL Doc is ideal for developers and DBAs who need to quickly produce high quality database documentation. SQL Doc integrates with SSMS, so you can document a database by right-clicking in the Object Explorer.
